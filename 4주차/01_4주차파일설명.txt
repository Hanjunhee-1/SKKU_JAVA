이번에는 class 와 객체에 대해서 배웠다.

솔직히 말하면 이미 C++ 과 JAVA 를 따로 공부하면서 잘 익혀뒀던 개념이라
어렵지 않았다.

※  코드가 은근 길어서 그냥 텍스트로만 간단하게 적어볼것이다!


작성해야 하는 코드 내용은 다음과 같다.

    1. Library 와 Book 이라는 클래스를 만든다.
    2. Book 에는 States 라는 변수가 존재하는데 해당 변수는 반납 또는 대여가 가능한지에 대한 여부를 판단하기 위함이다.
    3. Library 에는 책을 빌리는 Borrow() 와 책을 반납하는 Return() 이라는 메소드가 존재한다.
    4. main() 함수를 위한 클래스를 따로 생성하지 말고 Library 클래스에서 main() 을 사용한다.

조금 당황했던 부분이 있었다면 4 의 내용이었다.
이렇게 JAVA 코드를 작성해본 적이 없었지만 어렵지 않게 할수있었다.
main() 함수 안에서 해당 main() 함수를 가지는 클래스의 객체를 만들수있었다.
